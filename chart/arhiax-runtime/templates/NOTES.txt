-------------------------------------------------------------------------------
  Accessing the gateway
-------------------------------------------------------------------------------
{{- if .Values.gateway.ingress.enabled }}
The gateway is exposed via Ingress:
{{- range .Values.gateway.ingress.hosts }}
  http{{ if $.Values.gateway.ingress.tls }}s{{ end }}://{{ .host }}
{{- end }}
{{- else if eq .Values.gateway.service.type "LoadBalancer" }}
The gateway is exposed via LoadBalancer. Get the external address with:

  kubectl get svc -n {{ .Release.Namespace }} {{ include "arhiax.gateway.fullname" . }} \
    -o jsonpath='{.status.loadBalancer.ingress[0].ip}'

{{- else }}
The gateway is exposed as a ClusterIP service. Port-forward to test:

  kubectl port-forward -n {{ .Release.Namespace }} \
    svc/{{ include "arhiax.gateway.fullname" . }} 8080:{{ .Values.gateway.service.port }}

  curl http://localhost:8080/healthz
{{- end }}
{{- end }}
